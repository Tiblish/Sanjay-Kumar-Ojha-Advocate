# Sanjay Kumar Ojha | Advocate Portfolio

Elite legal portfolio for Adv. Sanjay Kumar Ojha, featuring high-fidelity React components, interactive jurisdictional mapping across India, and strategic legal insights.

## Features
- **Interactive Jurisdiction Map**: Precision-mapped court locations across India.
- **Strategic Practice Areas**: Detailed insights into Bail, Constitutional, Civil, and Service matters.
- **Lead Capture**: Seamless WhatsApp and callback integration for consultations.
- **BCI Disclaimer**: Compliance-first entry gate for legal ethics.

## Tech Stack
- React 19
- Vite 8
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Deployment
Automatically deployed via Vercel.
